## Set Nickname Discord Bot
## Install Python Environment
```sh
python -m venv venv
```
## Activate Python Environment
`Windows`
```sh
. venv/Scripts/activate
```
`Linux`
```sh
. venv/bin/activate
```
## Install Python Libraries
```sh
pip install -r requirements.txt
```

## Run Script
```sh
python main.py
```
## Command on Discord
```sh
!setnick <enter your desire nickname>
```
